# RAG Systems

This directory collects retrieval-augmented generation systems and reference
implementations that are useful for physics research.

RAG systems are larger than individual agent skills: they usually include
ingestion pipelines, indexing code, retrieval logic, model backends, and
runtime dependencies. They belong here when they demonstrate reusable patterns
for grounded scientific question-answering or literature-aware research tools.
