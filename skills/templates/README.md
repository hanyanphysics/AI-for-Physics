# Skill Templates

These templates help contributors create new skills with a consistent structure.

They are intentionally lightweight. Copy a template into a new skill directory,
rename the placeholders, and add only the sections that are useful for that
skill.

## Templates

| File | Purpose |
| --- | --- |
| [`SKILL.md.template`](SKILL.md.template) | Starting point for instructions used by compatible agents. |
| [`metadata.yaml.template`](metadata.yaml.template) | Starting point for structured skill metadata. |
